## Мои проекты

### E-commerce
**Ноутбук с кодом:** [E-commerce](https://github.com/Jactaesx/E-commerce_proj/blob/main/E-commerce_analysis.ipynb) 
**Описание:** Проект направлен на предоставление ответов на ключевые вопросы, касающиеся поведения пользователей, динамики заказов, популярности продуктов и активности пользователей.
**Навыки:** EDA, визуализация данных, когортный анализ, RFM-сегментация.
**Стек инструментов:** Python, Pandas, Numpy, Seaborn, Matplotlib.


### AB-test delivery app
**Ноутбук с кодом:** [AB-test_delivery_app](https://github.com/Jactaesx/AB_test_delivery_app/blob/main/AB_delivery_case.ipynb)
**Описание:** Командой проекта внедрила в приложение умную систему рекомендации товаров – предполагается, что такая система поможет пользователям эффективнее работать с приложением и лучше находить необходимые товары. Чтобы проверить эффективность системы рекомендаций, был проведен АБ-тест. Необходимо оценить, смогла ли новая система рекомендаций принести пользу бизнесу и пользователям приложения.
**Навыки:** Обработка данных, EDA, визуализация данных, проверка гипотез, тест Шапиро-Уилка, t-test, bootstrap
**Стек инструментов:** Python, Pandas, Numpy, Seaborn, Matplotlib, SciPy.Stats, Pingouin

### Online Cinema
**Ноутбук с кодом:** [Online_Cinema](https://github.com/Jactaesx/Online_cinema/blob/main/cinema.ipynb)
